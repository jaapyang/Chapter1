#help you,help me

最近回顾了我近10年的职场体验，有很多感触；在过去的十年里，我很有幸遇到很多帮助我进步与提升的朋友与同事，让我能在十年的时间里，取得了还算满意的成就与进步。但想想现在的大学生就业情况，我觉得，是时候去帮助他们了，我希望能组织一个团队，把我们在职场多年的工作经验与体会、技巧等，分享给需要帮助的人们。我相信这也是当年帮助我的人们所期望见到的。你是我最好的朋友，我希望有幸能邀请你一起参与到这么一件有意思的事情中，帮助曾经像我们一样迷茫的职场人或准职场人，就像由汤姆*克鲁斯主演的电影《甜心先生》（又名：Jerry Maguire ）里说的那样：”help you,help me“,在帮助你的同时，也帮助我。
